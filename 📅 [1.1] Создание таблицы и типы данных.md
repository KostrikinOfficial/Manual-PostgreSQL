# `📅 [1.1] Создание таблицы и типы данных`
___
## `🧩 [1.0.1] Самые распространенные типы данных`
| Тип | Что хранит | Пример |
|-----|------------|--------|
 | INTEGER | Целые числа | 25 |
 | BIGINT | Очень большие целые числа | 500000000000 |
| NUMERIC | Дробные числа | 12.5 |
 | VARCHAR(n) | Текст ограниченной длины | 'Андрей'
| TEXT | Текст неограниченной длины | 'Описание товара'
| BOOLEAN | Логическое значение | true |
| DATE | Дата | '2026-07-16' |
| TIMESTAMP | Дата и время | '2026-07-16 14:30:00' |
___
## `📅 [1.0.2] Что такое таблица`
### Мы уже знаем, что таблица состоит из строк и столбцов.
### Но пока таблица ещё нет, её нужно создать.
### Для этого существует команда: `CREATE TABLE`
___
## `🖊️ [1.0.3] Синтаксис CREATE TABLE`
### Самый простой пример:
    CREATE TABLE users (
        id INTEGER,
        name VARCHAR(100),
        age INTEGER
    );
### Разберем по частям:
### `CREATE TABLE` - Создать новую таблицу.
### `students` - Имя таблицы.
### Скобки `( );` - Внутри указывается содержимое
### Каждый столбец состоит из двух частей:
### `id` - название.
### `INTEGER` - тип данных.
### И в другом столбце:
### `name` - название.
### `VARCHAR(100)` - тип данных.
___
## `❓ [1.0.4] Что получится`
### После выполнения команды, PostgreSQL создаст пустую таблицу.
### Она будет выглядеть примерно так:
    id               | name               | age
    _________________|____________________|______
                     |                    |
                     |                    |
                     |                    |
### Строк пока нет, потому что мы создали таблицу, а не заполняли её.
___
