# 📅 [1.2] Создание таблицы
___
## [1.2.1] Создание таблицы
### `CREATE TABLE` - команда для создания таблицы
    CREATE TABLE IF NOT EXISTS название_таблицы (

        название_столбца тип_данных доп_условие

    );
### Пример:
    CREATE TABLE IF NOT EXISTS users (
    
        user_id SERIAL4 PRIMARY KEY,
        name TEXT NOT NULL,
        date TIMESTAMP DEFAULT NOW()

    );
### Разбор:
### `IF NOT EXISTS` - необязательное условие. Таблица будет создана, если она не существует.
### `PRIMARY KEY` - первичный ключ (задание уникального идентификатора)
### `NOT NULL` - поле не может быть пустым
### `DEFAULT` - значение по умолчанию. `NOW()` - текущая дата/время
