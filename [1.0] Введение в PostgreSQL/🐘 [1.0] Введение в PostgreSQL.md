# 🐘 [1.0] Введение в PostgreSQL
___
## 🐘 [1.0.1] PostgreSQL
### `PostgreSQL` - это объектно-реляционная Система Управления Базами Данных(СУБД).
___
## 🗂️ [1.0.2] База данных
### `База данных` - это набор взаимосвязанных данных.
### База данных может содержать множество таблиц и связи между ними.
___
## 📄 [1.0.3] Схема
### В одной базе данных можно хранить множество объектов.
### Чтобы их удобно организовывать, используются схемы.
### `Схема` - это пространство имен, которое помогает логически группировать таблицы, представления и другие объекты.
### По умолчанию большинство начинающих работают со схемой `public`
___
## 📅 [1.0.4] Таблица
### `Таблица` - это структурированный объект, в котором хранятся данные.
### Таблица состоит из:
### 1) Столбцы - описывают какие данные можно хранить
### 2) Строки - содержат конкретные записи
